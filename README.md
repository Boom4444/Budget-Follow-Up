# Budget Foyer — PWA

Application de suivi de budget personnel pour un foyer de 2 personnes.

**Accès direct :** https://boom4444.github.io/Budget-Follow-Up/

## Stack

React 18 · Vite · TypeScript · Tailwind CSS · Recharts · Zustand · Vite PWA

## Fonctionnalités

- Tableau de bord : graphiques mensuel et par catégorie, filtres année/mois/personne
- Saisie avec suggestions intelligentes (modèles récurrents + historique)
- 25 catégories, badge incompressible, 12 devises avec conversion automatique
- Dépenses récurrentes avec charge mensuelle estimée
- Fonctionne hors-ligne, installable sur iPhone via "Ajouter à l'écran d'accueil"

## Développement local

```bash
cd pwa
npm install
npm run dev
```

## Déploiement

GitHub Actions déploie automatiquement sur GitHub Pages à chaque push.
